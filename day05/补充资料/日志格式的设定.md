# 日志格式的设定

# 1.Format格式大全

| 占位符         | 含义                                                         |
| -------------- | ------------------------------------------------------------ |
| `%(name)s`     | 当前日志记录器的名称（比如你用 `logger = logging.getLogger("myLogger")`） |
| `%(filename)s` | 当前运行的脚本文件名（不包含路径）                           |
| `%(pathname)s` | 当前运行脚本的完整路径                                       |
| `%(funcName)s` | 打印日志的函数名                                             |
| `%(lineno)d`   | 打印日志所在的代码行号                                       |
| `%(module)s`   | 脚本模块名（去掉 `.py` 后缀的文件名）                        |
| `%(process)d`  | 当前进程的ID                                                 |
| `%(thread)d`   | 当前线程的ID                                                 |

# 2. 举例说明

```python
import logging

logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(filename)s - %(name)s - Line:%(lineno)d - %(message)s'
)

logger = logging.getLogger("MyCustomLogger")

def a_log():
    logger.info("这是一个测试日志")

a_log()

```

输出结果

```python
2025-07-30 10:42:31,456 - INFO - my_script.py - MyCustomLogger - Line:9 - 这是一个测试日志

```
## 注意：

多次调用 basicConfig(）——》只在主程序调用一次
